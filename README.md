# CSV Analyzer UI

A modern Python desktop application for analyzing CSV files, visualizing data, and exporting graphs to PDF.

## Features
- Select and analyze CSV files
- Visualize data with interactive graphs
- Filter data (e.g., by date range)
- Export graphs to PDF
- Modern, minimal UI/UX

## Setup
1. Install Python 3.8+
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Run the application:
   ```
   python src/main.py
   ```

## Requirements
- Python 3.8 or higher
- See requirements.txt for dependencies
